# C) Outreach message templates

## C1. Informational chat (short)
Hi [Name] — I’m exploring roles in [target role]. I noticed your work on [specific thing].  
I’m building [your project] and would love to ask 2–3 questions about [topic].  
Would you be open to a 10–15 min chat sometime this week? Thanks!

## C2. Referral ask (after rapport)
Hi [Name] — thanks again for your advice. I applied to [role] at [company].  
If you feel comfortable, would you consider referring me? I can share a 1–2 paragraph summary and my resume.

# D) Interview prep checklists

## D1. Coding interview checklist
- [ ] Clarify input/output and constraints
- [ ] Propose approach + complexity
- [ ] Code cleanly (names, structure)
- [ ] Test with 3 cases (normal, edge, stress)
- [ ] Discuss tradeoffs and alternatives

## D2. System design checklist
- [ ] Requirements + non-goals
- [ ] APIs
- [ ] Data model
- [ ] High-level diagram
- [ ] Scaling (read/write, caching, queues)
- [ ] Reliability (retries, timeouts, idempotency)
- [ ] Observability
- [ ] Security (auth, abuse)

## D3. Behavioral checklist (STAR)
- [ ] Situation & stakes
- [ ] Task/responsibility
- [ ] Action (what you did)
- [ ] Result (numbers if possible)
- [ ] Learning + prevention

# E) Weekly job search cadence

## Weekly plan (repeat)
- [ ] 10 applications (quality > quantity)
- [ ] 10 networking messages
- [ ] 3 mock interviews (1 coding, 1 system, 1 behavioral)
- [ ] 1 portfolio improvement (docs/tests/demo)
- [ ] 1 retrospective (what changed in the funnel)

# F) RAG ingestion tips (optional)
- Keep sections short and self-contained.
- Prefer lists, templates, and explicit step-by-step flows.
